# 快速开始指南

## 安装步骤

1. **安装依赖**
```bash
npm install
```

2. **启动开发环境**
```bash
npm run dev
```

这将自动：
- 编译主进程代码（TypeScript → JavaScript）
- 启动 Vite 开发服务器（React前端，端口 5173）
- 等待 3 秒后启动 Electron 应用窗口

**停止应用：**
- 在终端按 `Ctrl+C` 停止 Vite 服务器
- 关闭 Electron 窗口

## 使用说明

### 创建播放列表
1. 点击左侧"新建"按钮
2. 输入播放列表名称
3. 点击"创建"

### 添加视频
1. 选择要编辑的播放列表
2. 点击"添加视频"按钮
3. 选择本地视频文件（支持MP4、WebM、MOV、AVI、MKV）
4. 视频文件会被自动复制到应用数据目录

### 设置视频标签
- 点击视频项的"动作"或"待机"按钮切换标签
- **动作视频**：播放一次后自动跳转（顺序模式）或等待点击（点动模式）
- **待机视频**：循环播放，直到用户操作

### 调整播放顺序
- 拖拽视频项左侧的"⋮⋮"图标调整顺序

### 播放控制
1. 选择播放列表后，视频播放器会显示在下方
2. 点击"▶"开始播放
3. 选择播放模式：
   - **顺序播放**：自动按顺序播放
   - **点动控制**：需要手动点击"下一个"

### 播放模式说明

#### 顺序播放模式
- 动作视频：播放完成后自动播放下一个
- 待机视频：循环播放，点击"下一个"后，当前循环结束再播放下一个

#### 点动控制模式
- 动作视频：播放完成后停止，点击"下一个"立即播放下一个
- 待机视频：循环播放，点击"下一个"后，当前循环结束再播放下一个

## 数据存储位置

- **Windows**: `%APPDATA%/eventplayer/`
- **macOS**: `~/Library/Application Support/eventplayer/`
- **Linux**: `~/.config/eventplayer/`

数据库文件：`data/eventplayer.db`
视频文件：`videos/`

## 故障排除

### 视频无法播放
- 确保视频格式受支持（MP4、WebM、MOV、AVI、MKV）
- 检查视频文件是否损坏

### 应用无法启动
- 确保已安装所有依赖：`npm install`
- 检查Node.js版本（建议v18+）

### 数据库错误
- 删除应用数据目录，重新启动应用（会丢失所有数据）

## 构建生产版本

```bash
# 构建
npm run build

# 打包
npm run package
```

打包后的文件在 `release/` 目录中。
