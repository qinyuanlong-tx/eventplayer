# EventPlayer - 视频播放器

一个功能强大的本地视频播放器，支持播放列表配置、智能播放控制和播放列表管理。

## 功能特性

### 1. 播放列表配置
- 创建、编辑、删除播放列表
- 添加视频文件到播放列表
- 为每个视频设置标签（动作/待机）
- 拖拽排序调整播放顺序

### 2. 播放控制
- **顺序播放模式**：自动按列表顺序播放
  - 动作视频：播放完成后自动播放下一个
  - 待机视频：循环播放，等待用户操作
- **点动控制模式**：手动点击"下一个"按钮触发
  - 动作视频：点击后立即播放下一个
  - 待机视频：点击后等待当前循环结束再播放下一个
- 播放/暂停控制
- 进度条控制

### 3. 播放列表管理
- 新建播放列表
- 编辑播放列表名称
- 删除播放列表（同时删除关联视频文件）
- 播放指定播放列表

## 技术栈

- **Electron**: 跨平台桌面应用框架
- **React**: UI框架
- **TypeScript**: 类型安全
- **SQLite (better-sqlite3)**: 本地数据库
- **Vite**: 构建工具

## 安装与运行

### 开发环境

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

这个命令会自动：
1. 编译主进程代码
2. 启动 Vite 开发服务器
3. 等待 3 秒后启动 Electron 窗口

**手动分步启动（可选）：**

如果需要更多控制，也可以分步启动：

步骤 1 - 启动 Vite 服务器：
```bash
npm run dev:react
```

步骤 2 - 在另一个终端中，先编译然后启动 Electron：
```bash
npm run build:electron
npm run start
```

### 构建应用

1. 构建生产版本：
```bash
npm run build
```

2. 打包应用：
```bash
npm run package
```

## 数据存储

- **数据库**: SQLite数据库存储在 `用户数据目录/data/eventplayer.db`
- **视频文件**: 视频文件存储在 `用户数据目录/videos/`

## 使用说明

1. **创建播放列表**：点击左侧"新建"按钮创建播放列表
2. **添加视频**：选择播放列表后，点击"添加视频"按钮选择本地视频文件
3. **设置标签**：为每个视频设置"动作"或"待机"标签
4. **调整顺序**：拖拽视频项调整播放顺序
5. **播放控制**：选择播放列表后，使用播放器控制栏进行播放控制

## 注意事项

- 视频文件会被复制到应用程序数据目录，删除播放列表时会同时删除关联的视频文件
- 支持的视频格式：MP4、WebM、MOV、AVI、MKV
- 建议仅上传有版权的视频文件
